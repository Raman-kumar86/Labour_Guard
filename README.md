# Labour Guard -- Labour Law Compliance Platform

## 📌 Overview

**Labour Guard** is a full-stack web platform designed to promote
**labour law awareness and compliance while protecting worker
privacy**.\
The system provides accessible information about worker rights,
workplace regulations, and ethical labour practices while minimizing
unnecessary intrusion into workers' personal space.

The project demonstrates how technology can support **responsible labour
management, transparency, and worker welfare**.

------------------------------------------------------------------------

## 🎯 Objectives

-   Spread awareness about **labour rights and workplace regulations**
-   Provide easy access to **labour law compliance information**
-   Encourage **ethical workplace practices**
-   Promote **worker dignity and privacy**
-   Use technology to improve **labour law education and accessibility**

------------------------------------------------------------------------

## ✨ Features

-   📚 **Labour Law Awareness** -- Information about important labour

## 🚀 Installation & Setup

Clone the repository

    git clone https://github.com/Raman-kumar86/Labour_Guard.git

Navigate to the project folder

    cd Labour_Guard

Run the project using a local server (XAMPP / WAMP / Apache) and open in
browser

    http://localhost/Labour_Guard
